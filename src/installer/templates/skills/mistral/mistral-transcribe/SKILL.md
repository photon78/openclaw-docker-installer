# Skill: mistral-transcribe

## Wann verwenden
- Wenn eine Audio-Datei transkribiert werden soll (mp3, wav, ogg, m4a, webm)
- Sprachnachrichten zu Text
- Meetings, Interviews, Notizen als Audio

## Verwendung
```bash
python3 ~/.openclaw/workspace/skills/mistral-transcribe/transcribe.py --input <pfad> [--lang de]
```

## Output
Transkribierter Text.
