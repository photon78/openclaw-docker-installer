# Skill: mistral-ocr

## Wann verwenden
- Wenn ein Bild oder PDF Text enthält der extrahiert werden soll
- OCR auf Rechnungen, Dokumente, Screenshots, Fotos mit Text
- Wenn `image`-Tool nur beschreibt statt Text zu extrahieren

## Verwendung
```bash
python3 ~/.openclaw/workspace/skills/mistral-ocr/ocr.py --input <pfad_oder_url>
```

## Output
Extrahierter Text als Markdown.
